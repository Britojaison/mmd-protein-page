# Frontend

This folder contains all frontend-related files for the World Protein Day application.

## Structure

- `components/` - React components (ProteinForm, MealPlan, page)
- `utils/` - Utility functions (protein calculations)
- `styles/` - CSS and styling files

## Components

- `page.js` - Main page component
- `ProteinForm.js` - Form for user input
- `MealPlan.js` - Meal plan display component

## Tech Stack

- React 18
- Next.js 14
- Tailwind CSS
