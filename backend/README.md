# Backend

This folder contains all backend-related files for the World Protein Day application.

## Structure

- `api/` - API route handlers
- `data/` - JSON data files

## API Routes

- `generate-meal-plan.js` - Generates personalized meal plans
- `save-user.js` - Saves user data
- `user-count.js` - Returns count of users

## Data Files

- `products.json` - Milky Mist product information
- `recipes.json` - Recipe database
- `users.json` - User data storage

## Tech Stack

- Next.js API Routes
- File-based JSON storage
